:before and :after is used to insert something in the same parent.

Example1

CSS:
.ex1:before{
 content:"I";
}
.ex1:after{
 content:"Rice";
}
HTML:
<div class="ex1" >Eat</div>
Output:
Eat

Example2

CSS:
.ex2{
 width:50px;
 height:50px;}
.ex2:before{
/*width and height donot works. width and height will be same as .ex2{width:50px; and height:50px;}*/
 content:"";
 background-color:green;
 border:1px solid blue;
 padding:5px;
 border-radius:16px;
}
.ex2:after{
/*width and height donot works. width and height will be same as .ex2{width:50px; and height:50px;}*/
 content:"";
 background-color:orange;
 border:1px solid blue;
 padding:5px;
 border-radius:16px;
}

HTML:
<div class="ex2" >Eat</div>
Output:
Eat
